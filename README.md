# Project Metrush (가제)

UE5로 만드는 빠른 템포의 3D 액션 로그배니아(로그라이크 + 메트로배니아). 지하철 노선도 모티프, Windows PC.

> 이화여대 게임 개발 동아리 KING 여름방학 단기 프로젝트 (2026.06.25 ~ 08.28)
> 목표: 약 10분짜리 한 run이 완결되는 **프로토타입** 완성 → "한 판 더?" / go-no-go 판단
> 폴리싱보다 코어 루프 재미 검증이 우선.
